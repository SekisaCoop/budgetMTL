# Development

## Requirements

Install globally `webpack` and `webpack-dev-server`.

    npm install -g webpack webpack-dev-server

Run npm install

    npm install

## Working

  * run `webpack-dev-server`
  * go to `http://127.0.0.1:8080/webpack-dev-server/index.html`
  * work in `src` directory.
